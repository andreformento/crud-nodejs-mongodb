# CRUD, Express and MongoDB

## To run

 - With Docker
  - Requirements
    - Docker
    - Docker Compose
  - Commands
    - `docker-compose build && docker-compose up`

 - Local (development)
  - Requirements
    - npmjs
    - NodeJS
    - MongoDB

  - Commands
    - `npm install`
    - `npm run dev`

 - Remote (production)
  - Requirements
    - npmjs
    - NodeJS
    - MongoDB

  - Commands
    - `npm install`
    - `npm start`

## To access
[http://localhost:3000](http://localhost:3000)
